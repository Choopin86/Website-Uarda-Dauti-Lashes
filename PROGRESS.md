## Non-obvious decisions worth knowing

- Featured homepage service is Professional Lash Courses (`service8`) ‚Äî deliberate pick, it's the differentiator and targets a second audience (students).
- Missing `media.json` refs are skipped, never fatal ‚Äî intentional (data is incomplete by design right now), not a bug to "fix" by adding fallback images.
- Copy in the JSON files listed under Pending #3 was Claude-drafted from `bio.json` facts only (no invented marketing claims) and still needs owner sign-off.
- About page's structural build was deliberately finished before any styling ‚Äî owner's call, to keep the JSON‚Üítransform‚Üírender build separate from the CSS pass. (The About page was originally built with Claude guiding step-by-step and the owner writing the code, as a learning exercise; that workflow has since ended ‚Äî Claude now writes code directly like on the rest of the site.)
- Full-bleed section backgrounds require the `max-width`/`margin: 0 auto` centering to live on a separate inner wrapper, never on the same element as the section's `background-color` ‚Äî otherwise the background gets clipped to the centered content width. Found and fixed on both `.about-header` and `.business-story` this session; worth checking for on any new section.
- Renderers must assign a `className` to every element they create (even plain text nodes like bio paragraphs) ‚Äî `renderAboutHeader.js` and `renderAboutBio.js` were missing this and had to be retrofitted; every other renderer (e.g. `renderHero.js`, `renderTrust.js`) already follows this convention.
